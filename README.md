Simple Rabbit MQ example
========================

A very trivial Rabbit MQ example using the Erlang client libs

### Getting started

1. install Rabbit MQ Server {Version} Ex: 3.3.4
2. clone rmq_simple
3. cd rmq_simple
4. make RABBITMQ_VERSION={Version} deps
5. make compile
6. make run
7. Enjoy :smile: !
